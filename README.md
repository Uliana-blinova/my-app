Установка и запуск

1.Клонирование проекта

```
  git clone https://github.com/Uliana-blinova/my-app.git
  cd my-app
```

2.Установка зависимостей
```
npm install
```

3. Сборка проекта
```
npm run build
```
4. Запуск в режиме разработки
```
npm run serve
```
После запуска откройте браузер по адресу: http://localhost:8080
